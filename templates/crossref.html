How to deposit

  1. Make sure each journal has its DOI prefix, member ID, and depositor identity set. CrossRef will reject deposits with placeholder emails.
  2. For each issue you want to register: assemble the issue (so each article has start_page/end_page), then download the issue XML from this page.
  3. Upload the XML at doi.crossref.org using your CrossRef admin account. DOIs are minted when CrossRef accepts the deposit.
  4. Per-article XML is also available from each article's page if you need to deposit articles individually.
